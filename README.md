# Avance 01 Pablo Valenciano

Este proyecto se trabajó con el acompañamiento de Codex para refinar y limpiar las hojas de estilo, entender con mayor claridad el uso de pseudo-clases como `:hover` y selectores basados en `:target` y, a partir de ahí, hacer la interfaz más interactiva. Ese apoyo fue especialmente útil en la sección **KEY COMPETENCIES & SKILLS**, donde no tenía claro cómo estructurar las listas y vínculos internos que permiten mostrar cada tarjeta. También aproveché la guía de Codex para implementar la animación del botón de descarga del PDF.

Todas las imágenes utilizadas fueron creadas por mí. Finalmente, exploré Netlify y vi que ahí podía subir mi página, lo que me facilitó subir el proyecto al repositorio, para reforzar mi manejo de GitHub, y lo desplegué la versión en línea aquí: https://portfoliopablovalenciano.netlify.app
